# Linux & FOSS Project (Python Focused)

## 📌 Overview
This project explores Linux and Free and Open Source Software (FOSS) with Python as the chosen software. It includes theoretical concepts and practical shell scripting tasks.

## 🛠️Technologies Used
- Linux (Ubuntu)
- Bash Shell Scripting
- Python

##  Scripts Included
1. System Identity Report
2. Python Package Inspector
3. Disk and Permission Auditor
4. Log File Analyzer
5. Open Source Manifesto Generator

##  Objectives
- Understand FOSS principles
- Learn Linux commands and scripting
- Perform system-level automation

##  Features
- System information display
- Package checking
- Disk and permission auditing
- Log analysis
- Manifesto generation
